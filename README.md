# Menstrual Guiding Web App (Frontend Prototype)

## About the Project
This project was developed during the 24-hour TensorV2.0 Hackathon at Udgham 2.0 as part of team AgniAQI.

It is a frontend prototype built using HTML to demonstrate a multilingual menstrual health guidance platform aimed at supporting women in Tier 2 and Tier 3 cities.

- Multilingual interface (Hindi, Telugu, English, Marathi)
